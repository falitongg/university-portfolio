# ZWA
Základy webových aplikací - The basics of web applications

The main objective of the course was to teach everyone who completes it how to program a dynamic website, both its client-side (HTML, CSS, JavaScript) and server-side (PHP, architecture). They wanted everyone to demonstrate an understanding of the basic features of web applications. Therefore, the use of any frameworks or libraries that obscure the basic features and problems of web applications was not permitted.
