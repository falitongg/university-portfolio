# <a href="https://cw.fel.cvut.cz/wiki/courses/b6b36dsa/start">Data Structures and Algorithms</a>
## Synopsis
This course covers algorithm complexity and estimation methods. Topics include mathematical induction, recursive algorithms, data structures, sorting, and searching. It also introduces NP-completeness.
## Study Goals
Students will learn to estimate algorithm complexity, prove claims via induction, compare solution efficiencies, and navigate NP-completeness.
## Prerequisites
Prior programming and math proficiency, plus a capacity for exact reasoning.
