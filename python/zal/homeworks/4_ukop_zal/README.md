# Detailed assignment
Gregory-Leibniz series

Create a function named leibnizPi with one parameter (determining the number of terms) that adds up the given number of terms of the Gregory-Leibniz series.
That is, calling the function leibnizPi(1) returns 4.0, leibnizPi(2) returns 2.666666, etc. 

Nilakantha series

Create a similar function nilakanthaPi (also with one parameter) that sums the series.
That is, calling the function nilakanthaPi(1) returns 3.0, nilakanthaPi(2) returns 3.166666, etc.
