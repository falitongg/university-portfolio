# <a href="https://cw.fel.cvut.cz/wiki/courses/b0b37nsi/start">Design of IoT systems </a>

## Synopsis
The course presents IoT as a complex system, addressing system design and detailed design of individual components, primarily from a software perspective. It teaches students the skills to acquire, transfer, process, and interpret data in a "sensor-to-cloud" style.

## P.S.
All work was performed on <a href="https://www.raspberrypi.com/products/raspberry-pi-pico-2/">Raspberry Pi Pico 2 W</a>.
